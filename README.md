# AI-Generated Text Detection

This project classifies whether a paragraph was written by a human or generated by AI. It compares traditional NLP, deep learning, and transformer-based models for binary text classification.

## Project Overview

The goal of this project was to detect AI-generated fantasy-style text using multiple modelling approaches:

- Logistic Regression with Bag-of-Words and TF-IDF features
- Mean Embedding Classifier
- BiLSTM Classifier
- Fine-tuned DistilBERT

The project evaluates how well classical machine learning and deep learning models perform on human vs AI text detection.

## Dataset

The dataset was created from two sources:

- **Human-written text:** paragraphs extracted from three fantasy novels
- **AI-generated text:** fantasy-style paragraphs stored in `AI_dataset.csv`

The final dataset was cleaned, shuffled, and split using stratified sampling.

| Split | Samples |
|---|---:|
| Training | 4,199 |
| Validation | 900 |
| Test | 900 |

### Dataset Format

| Column | Description |
|---|---|
| `text` | Paragraph used as model input |
| `label` | Binary label: `0` = human-written, `1` = AI-generated |
| `source` | Source type, such as human or machine-generated |

> Note: Dataset files are not included in this repository because the human-written samples were extracted from copyrighted novels.

## Models Used

### Traditional Machine Learning

- Binary Bag-of-Words + Logistic Regression
- TF-IDF unigrams + Logistic Regression
- TF-IDF unigrams and bigrams + Logistic Regression

### Deep Learning

- Mean Embedding Classifier
- BiLSTM Classifier
- DistilBERT Sequence Classifier

## Results

| Model | Test Accuracy | Test F1 |
|---|---:|---:|
| DistilBERT | 1.0000 | 1.0000 |
| TF-IDF + Logistic Regression | 0.9989 | 0.9989 |
| Binary Bag-of-Words + Logistic Regression | 0.9978 | 0.9978 |
| BiLSTM | 0.8978 | 0.9002 |
| Mean Embedding Classifier | 0.7289 | 0.7319 |

DistilBERT achieved the best overall performance, while TF-IDF with Logistic Regression also performed extremely well.

## Key Findings

- Traditional NLP methods such as TF-IDF and Logistic Regression were highly effective for this dataset.
- DistilBERT achieved the strongest performance due to its contextual language understanding.
- BiLSTM performed better than the Mean Embedding model because it could capture word order and sequence patterns.
- The very high scores suggest that the dataset may contain strong differences between human-written and AI-generated text.

## Limitations

The results are very high, which suggests that the dataset may contain clear lexical or stylistic differences between the two classes. For a more realistic AI text detection system, the models should be tested on more diverse genres, unseen authors, and newer AI-generated text.
